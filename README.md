## 🚀 Deploy

Este projeto é um site estático puro (HTML + CSS + JS). Compatível com:

- **GitHub Pages** — basta ativar nas Settings do repositório
- **Netlify** — arrastar a pasta ou conectar ao GitHub
- **Vercel** — `npx vercel` na raiz
